# 客製化折線圖

這是 Windows/macOS 桌面版工具，可快速製作客製化折線圖。支援 Excel 貼上、多序列、區間色帶、圖片匯出，適合一般使用者。

**English guide:** [README.md](README.md)

## 下載

1. 到 GitHub 專案頁右側的 **Releases**。
2. 下載對應作業系統的安裝檔：
   - Windows：`NTOU_Tools_Setup.exe`
   - macOS：`NTOU_Tools.dmg`
3. 安裝並開啟：
   - Windows：雙擊 `.exe`。
   - macOS：開啟 `.dmg`，把 App 拖到 **Applications** 後開啟。

## 使用方式

1. 直接雙擊 `.exe` 檔案開啟程式。
2. 將 Excel 數據貼到「Excel 貼上」區塊，按「從 Excel 貼上套用」。
3. （可選）設定 X/Y 軸單位、顏色或區間色帶。
4. 按「繪製」預覽圖表。
5. 按「下載圖片」儲存圖片。

```
cm-1(X)	T(Y)
3997.43665	1.0051
3996.01357	1.00517
3994.59049	1.00465
3993.16741	1.00394
3991.74432	1.00349
3990.32124	1.00318
```

## 使用小技巧

- 多條線建議勾選「自動配色」。
- 製作簡報建議使用「A4 橫式」匯出。
- 匯出圖片預設為 PNG。
